# NewsIntel - 新闻智能分析平台（桌面版）

实时收集权威媒体新闻，智能分析含金量，帮助普通人快速掌握关键情报。

## 功能特点

- 📰 **实时新闻收集**：自动抓取人民日报、新华社、财新网等权威媒体
- ⭐ **智能含金量评分**：基于来源权威性、内容独特性、市场影响力等维度评分
- 🔍 **多维度筛选**：按分类、影响等级、含金量筛选
- 🖥️ **桌面应用**：独立窗口，系统托盘，开机自启
- 🔄 **自动更新**：每5分钟自动刷新新闻

## 安装步骤

### 1. 安装 Node.js

访问 https://nodejs.org 下载并安装 LTS 版本

### 2. 安装依赖

在项目目录打开命令行（PowerShell 或 CMD），运行：

```bash
npm install
```

### 3. 运行应用

```bash
npm start
```

### 4. 打包成安装包（可选）

生成 Windows 安装程序：

```bash
npm run build:win
```

打包完成后，在 `dist` 目录下找到安装程序。

## 使用说明

### 界面说明

- **顶部栏**：显示当前时间、刷新按钮、状态指示
- **统计栏**：显示新闻总数、关键新闻数、平均含金量、突发新闻数
- **左侧筛选**：按分类、影响等级、含金量筛选
- **右侧列表**：新闻卡片列表

### 操作说明

- **查看详情**：点击任意新闻卡片
- **刷新新闻**：点击顶部"刷新"按钮或托盘菜单"刷新新闻"
- **筛选**：勾选左侧筛选条件
- **排序**：点击"含金量"或"时间"按钮
- **查看原文**：在详情弹窗中点击"查看原文"

### 系统托盘

应用最小化后会在系统托盘显示图标：
- 单击图标：显示/隐藏主窗口
- 右键菜单：刷新新闻、退出
